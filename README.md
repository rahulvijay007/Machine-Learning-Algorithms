# Telecommunications Customer Classification

This script performs classification of telecommunications customers into different categories using the K-Nearest Neighbors (KNN) algorithm.

## Description
The script loads a dataset of telecommunications customers and performs data analysis, exploratory data analysis (EDA), data preprocessing, and KNN modeling for classification. It includes steps for standardization of variables and model evaluation.

## Usage
Execute the script in a Python environment with the required libraries installed. Follow the instructions and comments provided within the script for step-by-step execution.

## Features
- Data loading from CSV file
- Exploratory data analysis (EDA) using seaborn and matplotlib
- Data preprocessing including one-hot encoding and standardization
- KNN modeling for classification
- Model evaluation using accuracy score

## Dependencies
- pandas
- seaborn
- matplotlib
- numpy
- scikit-learn

## Dataset
The dataset `teleCust1000t.csv` contains information about telecommunications customers.
